Modélisation numérique¶

Expression analytique¶

\begin{equation} I(x) = sinc^2 (Bx)[1+cos(2Ax)] \quad avec: \ A = \pi a /\lambda D \ et \ B = \pi b / \lambda D \end{equation}

$ b $ reprsente la largeur des fentes, $ a $ reprsente la distance entre les fentes, $ D $ est la distance de l'cran au plan des fentes et $ \lambda $ est la longueur d'onde de la lumire incidente monochromatique.

Les valeurs de $ \Delta S $ et $ \Delta s $ sont dtermines thoriquement partir de $ b $ et $ a $: \begin{equation} \Delta S=2\lambda D/b \quad et \quad \Delta s=\lambda D/a \end{equation}

Définir une fonction Young(): tout le code en un bloc¶

In [14]:
def Young(a = 0.25, b = 0.15, D = 10):
    L = 30e-2   # Largeur de l'ecran
    x = linspace(-L/2, L/2, 200) 
    Lambda = 630e-9   
    # Variables intermidires:
    A = (pi*a*1e-3)/(Lambda*D); B = (pi*b*1e-3)/(Lambda*D)
    T1 = (sin(B*x)/(B*x))**2   # Terme de diffraction
    T2 = 1 + cos(2*A*x)        # Terme d'interfrence
    I = T1 * T2
    plot(x, I/max(I), x, T1, '--',lw =2)
    
    # Tache centrale
    DS = (2*Lambda*D)/(b*1e-3); Ds = (Lambda*D)/(a*1e-3)
    print(" Largeur de la tache centrale = %.3f m"%DS)
    print(" Largeur  de l'interfrange = %.3f m"%Ds)

def Young2D(a = 0.25, b = 0.15, D = 10):
    L = 30e-2   # Largeur de l'ecran
    x = linspace(-L/2, L/2, 200) 
    y = x
    X,Y = meshgrid(x,y)
    Lambda = 630e-9   
    # Variables intermidires:
    A = (pi*a*1e-3)/(Lambda*D); B = (pi*b*1e-3)/(Lambda*D)
    T1 = (sin(B*X)/(B*X))**2   # Terme de diffraction
    T2 = 1 + cos(2*A*X)        # Terme d'interfrence
    I = T1 * T2
    imshow(I, cmap="gray", interpolation='bilinear',
              origin='lower',vmin=I.min(), vmax= 0.1*I.max())
